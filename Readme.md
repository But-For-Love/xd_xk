# 西电自动选课工具

> **警告：该工具完全不能使用，一堆 Bug，主播自己双创周选课就被坑惨了，建议看看原仓库。**

西安电子科技大学自动选课工具。通过模拟 HTTP 请求与 [xk.xidian.edu.cn](https://xk.xidian.edu.cn) 交互，实现自动登录、验证码识别、选课、退课、捡漏。

---

## 快速开始

### 方式一：直接运行 exe（推荐）

从 Releases 下载 `西电自动选课工具.exe`，双击运行，无需 Python 环境。

### 方式二：源码运行

```bash
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple -r requirements.txt
python gui.py
```

## 功能一览

| 功能 | 说明 |
|---|---|
| **选课** | 逐门执行选课，课程已满时每秒自动重试 |
| **退课** | 逐门执行退课，自动重试 |
| **容量检查** | 持续监控选课池中的课程，有空位自动抢 |
| **捡漏** | 持续刷新全量课程列表，所有未满课程自动抢 |
| **浏览课程** | 从服务器拉取课程列表，支持搜索和勾选添加 |
| **未满过滤** | 浏览课程时可一键过滤掉已满课程 |
| **验证码 OCR** | 基于 ddddocr 自动识别验证码，也支持手动输入 |

## GUI 使用指南

### 界面布局

| 区域 | 功能 |
|---|---|
| **左侧边栏** | 账号配置、选课批次、验证码模式、调试开关 |
| **右侧选课池** | 当前课程列表，支持增删改查 |
| **操作栏** | 选课 / 退课 / 容量检查 / 捡漏 / 停止 |
| **底部日志区** | 实时显示操作结果 |

### 选课池管理

- **手动添加**：点击「＋ 添加课程」，填写课程号和课序号
- **浏览添加**：点击「📋 浏览课程」，从服务器拉取课程列表勾选
- **添加未满课程**：点击「⚡ 添加未满课程」，自动过滤已满课程，一键勾选
- **双击编辑**：双击选课池中的课程可修改信息
- **全选 / 全不选**：控制课程是否参与自动选课

> **必修课必须填课序号**（小卡片上 `[01]` 这类编号），选修课不需要。

### 四种执行模式

**1. 选课 / 退课**

对选课池中的课程逐门执行。勾选「连续重试」后，课程已满时会每秒重试，直到选上或与已选课程冲突。

**2. 容量检查**

持续监控选课池中的课程，每 0.5 秒检查一次，发现空位立即选课。适合已知目标课程、等待退课腾位的场景。

**3. 捡漏**（新增）

选择课程类别（必修 / 选修），持续刷新全量课程列表（5~10 秒一轮），所有未满课程自动尝试选课。适合错过选课时间、需要捡漏的场景。

- 选择类别后点击「🎯 捡漏」
- 自动跳过已选过的课程
- 日志实时显示每轮检查结果和已抢门数
- 随时点击「■ 停止」中断

## 命令行模式

```bash
python xk_main.py
```

编辑 `xk_main.py` 底部 `if __name__` 块切换功能：

| 函数 | 用途 |
|---|---|
| `main()` | 按 conf.json 执行选课 |
| `add_1(cat, courses, always)` | 编程式选课，可传自定义课程列表 |
| `del_1(cat, courses, always)` | 编程式退课 |
| `check()` | 课容量轮询监控，有余量自动抢 |

## 配置文件

GUI 模式下无需手动编辑 `conf.json`，关闭窗口时自动保存。手动编辑时的格式：

```json
{
  "ocr_captcha": "1",
  "debug": "0",
  "batch_name": "第一轮正选（国际创新周）",
  "bx_or_xx": 0,
  "bx": [
    { "KCH": "TE204003", "KXH": "02" }
  ],
  "xx": [
    { "KCH": "FL006066" }
  ],
  "data": {
    "loginname": "你的学号",
    "password": "你的密码",
    "captcha": "xxxx",
    "uuid": "xxxx"
  }
}
```

| 字段 | 说明 |
|---|---|
| `ocr_captcha` | `"1"` 自动识别验证码，`"0"` 手动输入 |
| `debug` | `"1"` 将 API 响应保存为 JSON 调试文件 |
| `batch_name` | 选课批次关键字，用于自动匹配当前开放批次 |
| `bx_or_xx` | `0` = 必修，`1` = 选修（GUI 自动管理） |
| `bx` | 必修课列表，每项需 `KCH` + `KXH` |
| `xx` | 选修课列表，每项只需 `KCH` |
| `data` | 登录凭据，留空则运行时提示输入 |

> **选课批次**：`batch_name` 不匹配时会自动列出所有可用批次，方便排查。

## 注意事项

- **不要**把含真实密码的 `conf.json` 上传到 GitHub
- 访问选课系统需要校园网或 VPN
- `check()` 里的课程号是硬编码的，用之前需要自行改代码
- exe 首次运行可能被 Windows Defender 拦截，点击「仍要运行」即可
